A vision board inspired by cork pin boards.

I wanted a way to display goals visually instead of using a traditional to-do list, so I built GoalCanvas. Each goal is pinned to a board, includes a hidden motivation that can be revealed on hover, and shows progress toward completion.

The project focuses on UI design, layout, and creating an experience that feels like a real physical board.

Features
Corkboard-style interface
Goal cards with categories
Hidden motivations
Progress tracking
Goal completion counter
Responsive layout
